This is todo list app, that i made as a course paper in my university. 
This app is really simple though: it allows you to add records, delete them, change them and synchronize all your progress via email.
The app uses sqlite database for synchronizing progress and storing any data.
Actually, i add some settings so you can change colors of background and text as you want

To be honest, it doesn't meant to be serious app and i'd like to do any android apps never more
